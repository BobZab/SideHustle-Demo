## üìñ About The Game

**SideHustle** is a dark comedy simulation game developed for the Western Colorado University Game Jam 2025.

While the theme of the jam was **"Critters,"** this project takes a morbid spin on the concept. Rather than befriending the local wildlife, you are tasked with a gritty entrepreneurial challenge. Catching, processing, and selling critters to save up money and keep the lights on.

It features a unique retro aesthetic and management mechanics set within a dingy apartment complex.

## üéÆ Gameplay Features

* **Dark Economy Loop:** Catch critters, process them at the **Butcher Station**, wrap the goods, and sell them at the **Sell Station** to increase your funds.
* **Upgrade System:** Use your hard-earned cash at the **Upgrade Station** to improve your equipment and efficiency.
* **Retro Aesthetics:** Built with a custom down-sampling shader to emulate the crunchy, pixelated look of 90s horror/PS1 games.
* **Atmospheric Lighting:** Strategic use of spot-lighting to create a tense, moody atmosphere.

## üöÄ How to Play / Installation

### To Run the Project in Godot:
1.  Clone this repository:
    ```bash
    git clone [https://github.com/YourUsername/SideHustle-Demo.git](https://github.com/YourUsername/SideHustle-Demo.git)
    ```
2.  Open Godot Engine (v4.x).
3.  Click **Import** and navigate to the `project.godot` file in the cloned folder.
4.  Press **F5** to run the debug build.

### Controls:
* **WASD:** Move
* **Mouse:** Look
* **E:** Interact (Stations/Items)
